⚠ Не удалось подключиться к серверу
Плагин требует локальный Python-сервер для генерации карт глубины.
  1. Откройте Терминал (Terminal)
  2. Перейдите в папку плагина:
    cd путь/к/3DRevealPro
  3. Запустите сервер:
    python server.py
  4. Дождитесь сообщения «Сервер готов»
  5. Нажмите Повторить ниже

Первый запуск может занять 1–3 минуты (загрузка модели).
